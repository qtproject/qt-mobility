Detailed Description

The LineReader class is a wrapper around a QIODevice that allows line-by-line reading.

This class keeps an internal buffer which it uses to temporarily store data which it has read from the device but not returned to the user.

Member Function Documentation

LineReader::LineReader ( QIODevice * device, QTextCodec * codec, int chunkSize = 1000 )

Constructs a LineReader that reads from the given device using the given codec. chunkSize is the number of bytes to read at a time (it is useful for testing but shouldn't otherwise be set).

bool LineReader::atEnd ()

Returns true if there are no more lines left for readLine() to return. It is possible for atEnd() to return false and for there to be no more data left (eg. if there are trailing newlines at the end of the input. In this case, readLine() will return an empty line (ie. position == selection).

QTextCodec * LineReader::codec ()

Returns the codec that the LineReader reads with.

int LineReader::odometer ()

How many bytes have been returned in the VersitCursor in the lifetime of the LineReader.

VersitCursor LineReader::readLine ()

Attempts to read a line and returns a VersitCursor describing the line. The cursor returned includes the data, as well as the position and selection index bounds. Data within those bounds represents the line. Data outside those bounds should not be used.
