Detailed Description

The QLandmarkCustomFilter class provides the interface which all custom filters should inherit.

All custom landmark filters should inherit off the QLandmarkCustomFilter and must implement the isMatch() function.

Custom filters inherently do not have native support and are thus may be subject to performance issues.
