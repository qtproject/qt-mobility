Detailed Description

The QOrientationSensor class is a convenience wrapper around QSensor.

The only behavioural difference is that this class sets the type properly.

This class also features a reading() function that returns a QOrientationReading instead of a QSensorReading.

For details about how the sensor works, see QOrientationReading.

See also QOrientationReading.
