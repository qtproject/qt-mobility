Multimedia

Multimedia provides a set of APIs that allow the developer to play, record and manage a collection of media content.

  • Overview
  • Reference documentation
    • Main classes
    • Classes for service implementers.

Overview

This library will require Qt 4.6.

To build the library, run qmake and make.
